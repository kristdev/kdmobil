================ Description ================

Ce module est un module d'inscription à une base de données pour newsletter
Il prend un charge un seul paramètre, le mail du souscrivant.
====================================================


Il necesite:
	- L'installation d'une base de données avec la table newsletter
		* Un fichier newsletter.sql est inclus avec le module

	- L'installation de la librairie xajax, indispensable pour le fonctionnement ajax
		* Suivre les instructions mentionées dans functions.php
